# HCC Single-Cell Atlas — Interactive Portal

Interactive web portal for the integrated HCC / Healthy liver single-cell
RNA-seq atlas described in:

> Díaz-Campos & Hernández-Lemus (2025). *Single-Cell Transcriptomic Profiling
> Reveals Immunometabolic Reprogramming and Cell-Cell Communication in the
> Tumor Microenvironment of Human Hepatocellular Carcinoma.*
> International Journal of Molecular Sciences (IJMS).

**Live portal:** https://hcc-atlas-portal.onrender.com

---

## Features

| Feature | Description |
|---------|-------------|
| UMAP explorer | 93,032 cells colored by cell type, condition, or gene expression |
| Cell type filter | Select one or multiple cell types |
| Condition toggle | All · Healthy Donors · HCC Diseased |
| Gene search | Violin plot + dot plot for any of 19,916 genes |
| Top-10 markers | Pre-computed per cell type (Wilcoxon, one-vs-rest) |
| Cell click | Click any UMAP point → top expressed marker genes |
| Download | Export filtered metadata as CSV |

## Local setup

```bash
# 1. Create environment
conda create -n hcc-portal python=3.11 -y
conda activate hcc-portal
pip install -r requirements.txt

# 2. Generate data files (requires full h5ad + scanpy)
conda activate sc_kernel
python prepare_data.py        # writes to ./data/  (~30–50 MB total)

# 3. Run portal
conda activate hcc-portal
python app.py                 # opens http://localhost:8050
```

## Deploy to Render.com

1. Push this directory to a GitHub repository
2. Upload `data/*.parquet` to Zenodo or GitHub Releases
   (files > 100 MB cannot be committed to GitHub directly)
3. Add a startup download step or use Render's persistent disk (configured in `render.yaml`)
4. Create a new Web Service on Render, connect the repo → auto-deploys
